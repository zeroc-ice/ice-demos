# Ice invoke

This demo illustrates the use of the Ice [streaming API].

## Building the Demo

```shell
make slice
swift build
```

## Running the Demo

First start the server:

```shell
.build/debug/Server
```

In a separate window, start the client:

```shell
.build/debug/Client
```

[streaming API]: https://archive.zeroc.com/ice/3.7/client-server-features/dynamic-ice/streaming-interfaces
