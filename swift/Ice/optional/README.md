# Ice optional

This demo illustrates the use of [optional class members] and [parameters].

## Building the Demo

```shell
make slice
swift build
```

## Running the Demo

First start the server:

```shell
.build/debug/Server
```

In a separate window, start the client:

```shell
.build/debug/Client
```

[parameters]: https://archive.zeroc.com/ice/3.7/the-slice-language/interfaces-operations-and-exceptions/operations
