# Ice context

This demo illustrates how to use [request contexts].

## Building the Demo

```shell
make slice
swift build
```

## Running the Demo

First start the server:

```shell
.build/debug/Server
```

In a separate window, start the client:

```shell
.build/debug/Client
```

[request contexts]: https://doc.zeroc.com/ice/3.7/client-side-features/request-contexts
